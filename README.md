# Don't Repeat Typing

Chrome 扩展，允许在任意可编辑输入中用 `/` 打开命令面板，快速插入预设多行文本。所有数据保存在浏览器本地 `chrome.storage.local`。

## 核心特性
- Slash 命令面板：`/` 唤出列表，支持 Arrow / PgUp / PgDn 循环选择、Enter 插入、Esc 关闭。
- 域名白名单：只在指定站点启用，避免干扰。
- 工具栏切换：按钮一键对当前站点启停。
- 命令管理：拓展选项页增删改命令，支持 JSON 导入导出。

## 安装
1. 打开 Chrome 地址栏输入 `chrome://extensions`。
2. 右上角打开 “开发者模式”。
3. 点击 “加载已解压的扩展程序”，选择本项目所在的文件夹即可。

## 使用方法
1. 在 Options 页添加命令、配置白名单域名。
2. 打开白名单中的站点，聚焦输入框。
3. 输入 `/`，用方向键挑选命令，按 Enter 插入。Esc 或点空白即可收起。也支持鼠标直接点击。
